Windows命令介绍ping网络诊断工具

2020-12-25 杂谈 206 字

ping主要是向特定的目的主机发送 ICMP（Internet Control Message Protocol 因特网报文控制协议）Echo 请求报文，测试目的站是否可达及了解其有关状态。

$ ping /？ #查看ping命令的帮助

Example

#最常用的格式
$ ping 114.114.114.114 #向IP地址发送请求报文

通过Wireshark捕获数据包发现，ping命令默认发送4个请求报文，源地址为PC地址，目的地址为指定的主机，默认TTL为128，数据包大小为32bytes（字节）

#改点参数看看
$ ping www.baidu.com -n 6 -l 64 -i 255

通过Wireshark捕获数据包发现，ping命令发送了6个请求报文，源地址为PC地址，目的地址被解析成为了网站IP地址，TTL变成了255，数据包大小变成64bytes（字节）
