Selenium自动化（1）Python开发环境配置

2022-11-17 杂谈 330 字

安装浏览器驱动

浏览器驱动是和浏览器对应的。 不同的浏览器需要选择不同的浏览器驱动。

目前主流的浏览器中，谷歌 Chrome 浏览器对 Selenium自动化的支持更加成熟一些。

  • 谷歌浏览器驱动下载，注意浏览器驱动 必须要和浏览器版本匹配。

  • 安装浏览器驱动

    把exe文件复制到浏览器的安装目录下：C:\Program Files (x86)\Google\Chrome\Application（根据自己实际安装目录）

  • 配置系统环境变量

    此电脑→右击属性→高级系统设置→环境变量→用户变量→Path→编辑→新建

    C:\Program Files (x86)\Google\Chrome\Application\

安装Selenium库

不同的编程语言选择不同的Selenium客户端库。

对应我们Python语言来说，Selenium客户端库的安装非常简单，用 pip 命令即可。

  • 打开 命令行程序，运行如下命令
pip install selenium
  • 如果安装不了，可能是网络问题，可以指定使用国内的豆瓣源
pip install selenium -i https://pypi.douban.com/simple/

测试环境

下面的代码, 可以自动化的打开Chrome浏览器，并且自动化打开百度网站。

# 测试Chrome驱动
from selenium import webdriver

driver = webdriver.Chrome()
url = 'https://www.baidu.com/'
driver.get(url)
driver.maximize_window()
