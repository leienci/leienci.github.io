JAVA开发 JDK环境配置

2021-09-04 杂谈 165 字

（一）JDK的安装

演示的版本为jdk-8u201-windows-x64，安装的系统为Windows10版本1909。

安装过程默认下一步即可

（二）PATH配置

1.关闭JAVA自动检查更新

2.验证JDK安装是否成功

& java -version #查询java版本

3.设置环境变量

控制面板\系统和安全\系统-高级系统设置

4.新建系统变量

变量名：JAVA\_HOME
变量值：C:\\Program Files\\Java\\jdk1.8.0\_201
变量名：CLASSPATH
变量值：.;%JAVA\_HOME%\\lib\\dt.jar;%JAVA\_HOME%\\lib\\tools.jar;

5.添加环境变量

%JAVA\_HOME%\\bin
%JAVA\_HOME%\\jre\\bin

6.验证配置

& javac

参考资料：https://www.runoob.com/w3cnote/windows10-java-setup.html
